\textbf{Message passing consists of three steps:}
\begin{enumerate}
    \item Computing messages ($\phi_{mes}$)
    \item Aggregating messages ($\phi_{agg}$)
    \item Updating node values ($\phi_{upd}$)
\end{enumerate}
